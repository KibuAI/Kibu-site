Workflow

상황에 딱 맞는 3가지 루틴

선생님의 상황에 맞춰 가장 효율적인 방법을 선택하세요.

루틴 A. 꼼꼼한 작성

활동지, 보고서 등 학생 근거 자료가 많을 때 추천합니다.

  • 1. 학생 선택 & 자료(PDF/사진) 드래그
  • 2. [외부자료 분석]으로 내용 추출
  • 3. 교사 코멘트(핵심) 1줄 작성
  • 4. [AI 프롬프트 조립] → 생성

루틴 B. 신속 일괄 작성

엑셀 설문 내용만으로 반 전체 초안을 빠르게 잡습니다.

  • 1. 엑셀 설문 결과로 프로젝트 생성
  • 2. 외부 자료 없이 코멘트에 힌트 작성
  • 3. 섹션 선택 후 [전체 학생 작성]

루틴 C. 기존 글 다듬기

이미 써 둔 글의 글자 수를 줄이거나, 문맥을 점검합니다.

  • 1. 기존 작성 문장 붙여넣기
  • 2. [글자수 줄이기]로 바이트 맞춤
  • 3. [문맥 개선]으로 최종 점검
