Core Features

생기부 업무 전체를 자동화합니다

단순 요약 AI가 아닙니다. 자료 분류부터 초안 생성, 규칙 검토까지 한 흐름으로 이어집니다.

One-Pass 맥락 분석

자료를 단순 OCR로 요약하지 않습니다. 자료 묶음을 한 번에 읽고 활동의 주제·진행 단계·학생의 고유한 기여를 종합적으로 구조화합니다.

분석 흐름 미리보기

자료 자동 배분

명렬표와 섞인 스캔본 하나만 올리면 학번 기준으로 학생별 보관함에 정확히 분류됩니다.

학생 자료 혼입 방지

여러 명을 동시에 처리해도 학생별로 완전히 분리된 공간에서 처리되어 자료가 섞이지 않습니다.

일괄 생성 · 규칙 검토

학급 전체 초안을 한 번에 생성하고 바이트 제한·기재 금지어를 자동으로 검토합니다.

검토 카드 치환

검증 이슈를 클릭하면 문제 문장이 하이라이트되고, 치환 버튼으로 즉시 수정됩니다.

로컬 개인정보 보안

모든 처리는 교사 PC에서 진행되며 API 키는 암호화되어 안전하게 보관됩니다.

Why KIBU

기존 방식

  • 학생별 자료를 일일이 취합하고 복사·정리
  • 한 명씩 반복해서 문장 작성·타이핑
  • 나이스 바이트 수동 계산과 미세 조정
  • 기재 금지어를 눈으로 하나씩 확인
  • 자료가 뒤섞일까 늘 신경 쓰임

KIBU와 함께

  • 명렬표 + 자료를 한 번에 학생별 자동 배분
  • 학급 전체 초안을 클릭 한 번에 일괄 생성
  • 바이트·금지어를 자동으로 검증
  • 근거 자료를 하이라이트하는 검토 카드
  • 학생별 자료가 완전히 분리되어 안전
