강력한 자동화 기능

KIBU AI가 선생님의 시간을 어떻게 아껴주는지 스크린샷으로 확인해 보세요.

Data Intake

자료 드래그 및 자동 배분

나이스 명렬표뿐만 아니라, 여러 학생이 섞여 있는 스캔본이나 설문 응답을 한 번에 가져오세요.

KIBU가 드래그 앤 드롭된 파일에서 학생별 이름과 학번을 인식하여 개인 보관함으로 자료를 정확하게 배분해 줍니다.

  • 자료 드래그 앤 드롭 등록
  • 설문 자료 원클릭 동기화
  • 학생별 보관함 자동 구성
  • 스캔본 배정 안내: AI가 학번과 이름을 찾기 어려운 여러 학생이 섞인 스캔본(통합 PDF)의 경우, 우측의 [배정 정보 수정] 기능을 이용하여 각 페이지를 특정 학생에게 쉽고 빠르게 수동 배정할 수 있습니다.
Custom Prompt

맞춤형 프롬프트와 관점 적용

단순한 요약이 아닌, 선생님의 교육 철학과 과목의 특성을 생기부에 담아냅니다.

수업 중 어떤 역량과 태도에 주안점을 두었는지, 학생들의 어떤 탐구 활동이나 성장 과정을 부각하고 싶은지 등 선생님만의 구체적인 평가 기준과 관점을 입력하여 한 차원 높은 초안을 생성하세요.

Batch Generation

학급 전체 일괄 생성

한 명씩 초안을 만들고 기다릴 필요가 없습니다.

자료 배분과 프롬프트 설정이 끝났다면 일괄 생성 버튼 하나로 전체 학생의 초안을 동시에 생성합니다.

Rule Management

빈틈없는 규칙 검증 및 내보내기

작성된 내용이 나이스 입력 규정을 위반하지 않도록 기재 금지어를 피하고 바이트를 자동으로 조절합니다.

검토 카드로 최종 점검을 마친 초안은 나이스 업로드에 최적화된 엑셀 파일로 일괄 저장됩니다.

Advanced Setup

모델 최적화 및 프롬프트 공유

상황에 맞게 OpenAI와 Gemini 모델을 선택하거나 초기화할 수 있습니다.

또한 성공적인 결과를 얻은 맞춤형 프롬프트는 동료 교사와 공유할 수 있어 학교 전체의 기록 수준을 높일 수 있습니다.
